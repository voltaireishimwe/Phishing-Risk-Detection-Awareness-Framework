Framework Architecture

Random Forest Classifier

Ensemble learning method using multiple decision trees with weighted bootstrap sampling for handling class imbalance in phishing emails.

Elastic Net Feature Selection

Combines L1 and L2 regularization for optimal feature selection from email content, headers, and URL features.

Weighted Bootstrap Sampling

Advanced sampling technique addressing class imbalance by assigning weights to minority class instances.

Extracted Features from Dataset

  • Sender/Receiver Analysis: Domain reputation, email patterns
  • Content Features: Suspicious keywords, urgency indicators
  • URL Features: Domain age, SSL certificates, redirects
  • Temporal Features: Sending time patterns, frequency
  • Technical Features: HTML content, attachments, headers
